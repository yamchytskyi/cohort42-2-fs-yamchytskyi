Образ (image) шаблон, основа для создания контейнера. Содержит необходимые настройки,
 программный код, зависимости. Представлен в виде набора абстрактных слоев, которые накладываются друг на друга
 и при необходимости перезаписываются

Dockerfile специальный текстовый файл с прописанными инструкциями (командами) по созданию образа

Контейнер это экземпляр приложения, созданный из образа.

Реестр это хранилище образов. Образы можно скачивать из реестра и на их основе созвавать (разворачивать) контейнеры
В реестр можно загружать новый или измененные образы для использования


docker ps выводит список запущенных контейнеров

docker ps -a выводит список всех контейнеров

docker build собирает образ из Dockerfile

docker images выводит все images

docker stop останавливает работу контейнера

docker rm удаляет  остановленный контейнер

docker rmi удаляет image
